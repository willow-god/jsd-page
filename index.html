介绍

该站点为自建的jsDelivr镜像，用于加速国内资源，采用Cloudflare中转回源，使用多吉云融合CDN分发资源。

由 LiuShen搭建并维护，你可以访问博客找到我的联系方式。

所有文件结构均遵循 jsDelivr 原有结构。

使用时，请遵守jsDelivr与npm的ToS，本镜像不允许加速图片等资源，若发现此类行为，将禁止违规者使用。

使用方法

Endpoint: https://jsd.liushen.fun

将 https://cdn.jsdelivr.net 替换为 https://jsd.liushen.fun 即可。

建议指定版本号加载，否则缓存将经过一年时间才会刷新。

// load any project hosted on npm

https://jsd.liushen.fun/npm/package@version/file


// load jQuery v3.6.4

https://jsd.liushen.fun/npm/jquery@3.6.4/dist/jquery.min.js


// use a version range instead of a specific version

https://jsd.liushen.fun/npm/jquery@3.6/dist/jquery.min.js

https://jsd.liushen.fun/npm/jquery@3/dist/jquery.min.js


// omit the version completely to get the latest one

// you should NOT use this in production

https://jsd.liushen.fun/npm/jquery/dist/jquery.min.js


// add ".min" to any JS/CSS file to get a minified version

// if one doesn't exist, we'll generate it for you

https://jsd.liushen.fun/npm/jquery@3.6.4/src/core.min.js


// omit the file path to get the default file

https://jsd.liushen.fun/npm/jquery@3.6

// add / at the end to get a directory listing

https://jsd.liushen.fun/npm/jquery/

使用规则

不要用于图片、字体文件的加速，这违反了 jsDelivr 的 TOS：https://www.jsdelivr.com/terms/acceptable-use-policy-jsdelivr-net
同样违反 npmjs.com 的 TOS: https://docs.npmjs.com/policies/open-source-terms#acceptable-content
若您有托管图片的需求，推荐使用他人搭建的公开图床（推荐去不图床 @ 杜老师说）。
若您有请求大字体文件的需求，请考虑 Google Fonts ，以及对应的国内镜像。
